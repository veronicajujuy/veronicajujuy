### ☕ Un poco sobre mí

Vengo del desarrollo backend en Java y Node.js, pero mi forma de trabajar siempre fue la misma: cuando aparece un problema, lo investigo desde la raíz hasta entenderlo — código, infraestructura o el flujo completo, no importa en qué capa esté.

Eso me llevó naturalmente a meterme en infraestructura cloud (Docker, MySQL, administración de servidores en producción), seguridad de aplicaciones (SAST, SCA, escaneo de imágenes Docker) y, más recientemente, a integrar herramientas de IA en mi trabajo diario: desde pipelines de seguridad hasta diseño de programas de formación.

Combino esto con la docencia, ayudando a futuros desarrolladores a construir proyectos reales en el mundo IT.

- Integración de LLMs en backend Java: endpoints que consumen la API de Anthropic para análisis y generación de texto.

#### 🔹 Migración Java 21 → Java 25 con OpenRewrite
**Stack:** Java · Spring Boot · Maven · OpenRewrite
- Guía práctica basada en casos reales: prerequisitos, ejecución de la receta de migración, errores frecuentes (Lombok, compiler plugin, JAVA_HOME) y matriz de versiones compatibles.
- [Ver repositorio →](https://github.com/veronicajujuy/Migracion-Java21-java25)

#### 🔹 Pipeline de datos de vuelos — AzureFlightPipeline
**Stack:** Azure Data Factory · Databricks · PySpark · Delta Lake · Arquitectura Medallion
- Pipeline end-to-end sobre vuelos comerciales en el norte argentino (Bronze → Silver → Gold).
- [Ver repositorio →](https://github.com/veronicajujuy/AzureFlightPipeline)

#### 🔹 Portfolio Cloud & Infraestructura
**Stack:** Docker · MySQL · Huawei Cloud
- Casos reales de diagnóstico y resolución de incidentes en producción, documentados como postmortems.
- Incluye recuperación de servidor con disco saturado por logs Docker, corrupción de socket MySQL e intento de intrusión vía webshell.
- [Ver repositorio →](https://github.com/veronicajujuy/cloud-infra-portfolio)

#### 🔹 Extensión "Tomar Asistencia" — Chrome Extension
**Stack:** JavaScript · Web Scraping · Automatización
- Automatiza el registro de asistencia en clases virtuales.
- Reducción del tiempo de 30 min a menos de 5 min por clase.
